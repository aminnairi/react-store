# Contributing

## Requirements

- Git
- Docker
- Docker Compose

## Clone the repository

```bash
git clone https://github.com/aminnairi/react-store.git ~/react-store
cd ~/react-store
```

## Install the Node dependencies

```bash
npm install
```

## Build the project

```bash
npm run build
```

## Generate the TypeScript types

```bash
npm run types
```

## Publish the project

```bash
npm publish --access public
```
